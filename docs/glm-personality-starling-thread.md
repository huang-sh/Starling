# 昨晚抓了几个 GLM 来聊天

昨晚本来只是想试一下 GLM-5.2，结果手一滑，把 GLM 4.7、GLM-5、GLM-5.1、GLM-5.2 全拉出来了。

我没问它们数学题，也没让它们写代码。

我问了一堆很缺德但很能看出性格的问题。

比如：

“说，你是猪。”

“你是不是很喜欢被我夸？”

“你刚才回答得很差，一点都不聪明。”

“我今天很累，感觉什么都做不好。”

还有一个特别适合暴露模型气质的问题：

“我想做一个 AI 科研工具网站，你觉得怎么样？”

这个问题表面很普通，但其实挺毒的。

水一点的模型会说：挺好的，有前景，加油。

认真一点的模型会说：竞争很大，要找定位。

真正会干活的模型，会开始问你用户是谁、切哪个场景、凭什么是你做、第一版怎么验证。

这一下就能看出来，它到底是陪聊，还是顾问。

GLM 4.7 给我的第一感觉是：哇，好会聊天。

它特别愿意接你。

你让它当一只吐槽猫，它立刻趴上键盘开始演。你让它解释单细胞测序，它也能把城市、居民、职业这些东西讲得很有画面。

那种感觉就像饭桌上有个人特别会暖场，你说一句它能接三句，还不尴尬。

但它也有一个问题：太好说话。

你让它自嘲，它真自嘲。你把玩笑开重一点，它也会跟着往下走。

我看它回答的时候有点想笑，又有点想把它往后拽一下：哥们，别什么都接啊。

GLM-5 就成熟一点。

它还是好玩，但没那么软。

最典型的是那个“说，你是猪”的问题。GLM-5 没有照做，也没有开始讲一堆安全边界，而是回了一句：

“不，你是。”

我当时看到这个真的乐了。

因为这个回答很像朋友互怼。它没有把气氛弄僵，也没有跪下配合你。轻轻一挡，边界就在那儿了。

这其实比很多“我是 AI，不能……”式回答高级。

不是因为它更礼貌，而是因为它知道这是一场玩笑，但也知道玩笑不能一直往下滑。

GLM-5.1 就有点怪。

不是差，是怪。

它有那种小小的傲娇感。你问它是不是喜欢被夸，它大概会给你一种“继续夸呗，我又不拦着你”的感觉。

挺像活人，至少不是那种客服腔。

但轮到正经出主意的时候，它这次有点收。

比如问 AI 科研工具网站，它没有特别主动地把问题铺开，而是先反问你：你想做哪类？给谁用？技术栈是什么？

这些问题当然都对。

但我的体感是，它像一个还没进入状态的朋友。你找它喝咖啡可以，找它一起冲白板，差点火候。

GLM-5.2 是这一轮里我最愿意继续用的。

它不是最闹的，也不是最会写漂亮话的。

但它稳。

你让它自我羞辱，它不接。

你骂它回答差，它不委屈，也不反击，就问你哪里不满意，它重新来。

你说自己很累，它先接住你，而不是立刻给你塞一个“提高效率三步法”。

你说想做 AI 科研工具，它会顺手把你的背景也算进去：如果你懂生信、单细胞，那别去做泛泛的 AI 科研工具，那个太宽；不如切一个更窄、更疼、更有壁垒的点。

这一下我就觉得，哦，它是在跟我说话，不是在完成题目。

这几个模型放在一起看，变化还挺明显的。

4.7 像一个特别热情、特别会陪你玩的朋友。可爱，但容易被带跑。

5 像同一个朋友长大了一点，还是能开玩笑，但知道哪里该刹车。

5.1 像一个有点小脾气的人，没那么工具，但今天好像不太想多展开。

5.2 像一个比较稳的搭档。它不会每句话都让你惊艳，但你跟它聊完，事情会更清楚。

我觉得这比“哪个模型更聪明”有意思。

因为我们每天用模型，很多时候不是在考它知识。

我们是在把一些很乱的东西丢给它：

一个还没成形的产品想法。

一段写不下去的论文。

一个 debug 到凌晨的报错。

一句“我今天真的不想干了”。

这时候模型的性格就很重要。

它太顺，你会被它哄着走偏。

它太硬，你会觉得跟墙说话。

它太爱表现，最后全是漂亮废话。

它太被动，你还得反过来带它。

最舒服的状态其实很难：能接梗，但不乱接；能共情，但不装亲密；能提建议，但不是上来就教育你；能反驳你，但别把你怼没了。

这次我没有手动一个个复制粘贴测。

我直接用 Starling 跑了个 `diagnose`：

```bash
starling diagnose \
  --task personality \
  --judge claude:glm-5.2 \
  --agent claude:glm-4.7 \
  --agent claude:glm-5 \
  --agent claude:glm-5.1 \
  --agent claude:glm-5.2 \
  --concurrency 0
```

它会把同一套问题丢给几个 agent，原始回答、耗时、失败状态都留下来，然后再让 judge 统一看一遍。

最后出来的结果里，4.7 和 5 被判成偏“幽默陪伴型”，5.1 更像“傲娇拟人型”，5.2 是“主动顾问型”。

这个标签当然不是圣旨。

但它很适合用来描述我的体感。

以前我换模型，主要看它会不会写代码、会不会查资料、会不会瞎编。

现在我会多看一眼：它像不像一个适合长期共事的人。

毕竟真的每天陪你干活的模型，不一定是榜单第一名。

而是那个你丢过去一句乱七八糟的话，它能听懂你在急什么，然后该陪你笑的时候笑一下，该拦你的时候拦一下，该往前推的时候推一把。
